Come Configurare un Web Server su Raspberry Pi 3B

Questa guida spiega come configurare un web server su un Raspberry Pi 3B utilizzando Apache2, PHP, MariaDB e MySQL.

Passo 1: Aggiornare e Aggiornare il Sistema

Prima di iniziare, assicurati che il tuo Raspberry Pi sia aggiornato. Esegui i seguenti comandi:

sudo apt update
sudo apt upgrade

Passo 2: Installare Apache2

Apache2 è il software del server web. Installalo utilizzando:

sudo apt install apache2 -y

Una volta installato, puoi verificare se Apache2 è in esecuzione visitando l'indirizzo IP del tuo Raspberry Pi in un browser.

Per maggiori informazioni su Apache2, visita il sito ufficiale: https://httpd.apache.org/

Passo 3: Installare PHP

PHP è un linguaggio di scripting lato server. Installalo con:

sudo apt install php libapache2-mod-php -y

Per testare PHP, crea un file chiamato info.php nella directory /var/www/html:

sudo nano /var/www/html/info.php

Aggiungi il seguente contenuto:

<?php
phpinfo();
?>

Salva il file e visita http://tuo-ip-pi/info.php nel browser.

Per scaricare PHP su altre piattaforme, visita: https://www.php.net/downloads

Passo 4: Installare MariaDB

MariaDB è un sistema di gestione di database. Installalo con:

sudo apt install mariadb-server -y

Proteggi l'installazione eseguendo:

sudo mysql_secure_installation

Per scaricare MariaDB su altre piattaforme, visita: https://mariadb.org/download/

Passo 5: Installare il Client MySQL

Per interagire con MariaDB, installa il client MySQL:

sudo apt install mysql-client -y

Per scaricare MySQL su Windows, visita: https://dev.mysql.com/downloads/installer/

Per scaricare MySQL su Linux, visita: https://dev.mysql.com/downloads/repo/apt/

Passo 6: Testare la Configurazione

Assicurati che Apache2, PHP e MariaDB funzionino insieme. Crea un file PHP per connetterti al database:

<?php
$servername = "localhost";
$username = "root";
$password = "your_password";

// Creare connessione
$conn = new mysqli($servername, $username, $password);

// Controllare connessione
if ($conn->connect_error) {
    die("Connessione fallita: " . $conn->connect_error);
}
echo "Connessione riuscita";
?>

Salva il file in /var/www/html e accedilo tramite il browser.

Conclusione

Hai configurato con successo un web server sul tuo Raspberry Pi 3B con Apache2, PHP, MariaDB e MySQL. Ora puoi ospitare i tuoi siti web e applicazioni!
